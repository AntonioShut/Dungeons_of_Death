# Dungeons_of_Death
Здесь представлено описание проекта Dungeons of Death. Это компьютерная игра в жанре средневекового фентези и пошаговой стратегии, где вы выбираете себе героя и играя за него зачищаете древнее подземелье которое прозвали "Подземелье Смерти".
Игра двумерная в 16 битном стиле. На начало игры будут представлены три класса Теневой Маг, Вор и Воин.
